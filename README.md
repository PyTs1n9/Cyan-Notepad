# BaiQingTodo

一款基于 **Tauri v2 + React + TypeScript** 构建的 Windows 桌面工具箱，集待办事项管理、富文本记事本、主题定制、中英文切换于一体。小巧轻量，开箱即用。

> 📦 前往 [Releases](../../releases) 页面下载最新版安装包（`.exe`），双击即可安装使用。

---

## 功能一览

### 📋 待办事项

- 创建、完成、删除任务
- 三级优先级（低 / 中 / 高），用颜色直观区分
- 筛选视图：全部 / 进行中 / 已完成
- 实时统计：显示总数与已完成数

### 📝 富文本记事本

- **双模式编辑**：所见即所得（WYSIWYG）与 Markdown 源码模式自由切换
- **MD 分屏预览**：左侧源码编辑 + 右侧实时预览，双向滚动同步
- **工具栏可隐藏**：一键收起工具栏，获得沉浸式编辑体验
- **丰富的排版能力**：
  - 标题 H1 / H2 / H3、段落、引用块、分隔线
  - 粗体 / 斜体 / 下划线 / 删除线 / 高亮
  - 有序列表 / 无序列表
  - 文字颜色（取色器）与字号（12px ~ 48px）
  - 文本对齐（左 / 中 / 右）
- **图片插入**：支持本地图片自动转 Base64 内嵌
- **标签系统**：编辑器中插入 `#标签`，侧栏按标签筛选笔记
- **自定义字体**：导入 `.ttf` / `.otf` / `.woff` / `.woff2` 字体文件
- **导入 Markdown 文件**：批量导入本地 `.md` 文件，自动创建笔记
- **自动保存**：编辑后 800ms 自动写入磁盘，无需手动保存
- **可调节侧边栏**：拖拽侧边栏右边框调整宽度（180px ~ 400px）

### 🎨 主题系统

- **4 套预设主题**：深色 / 蓝调 / 忧郁黄 / 清新绿，一行切换
- **自定义调色板**：5 个颜色字段（主背景、次背景、侧边栏、主文字、强调色）自由搭配
- **保存预设**：最多保存 5 个自定义配色方案，支持命名管理与一键应用

### 🌐 国际化

- 中文 / English 双语言切换，全界面文本覆盖

## 安装与使用

### 安装

1. 前往 GitHub [Releases](../../releases) 页面下载最新版 `.exe` 安装包
2. 双击运行安装程序，按提示完成安装
3. 从开始菜单或桌面快捷方式启动 BaiQingTodo

### 待办事项

1. 应用默认进入「待办事项」页面
2. 在顶部输入框输入内容 → 选择优先级（低/中/高）→ 点击「添加」或按 `Enter`
3. 点击任务左侧圆圈标记完成（再次点击可恢复为未完成）
4. 鼠标悬停任务时，右侧出现删除按钮
5. 顶部筛选标签可切换「全部 / 进行中 / 已完成」视图

### 记事本

1. 在左侧导航栏点击「记事本」进入笔记页面
2. 点击「新建笔记」创建空白笔记，或点击「导入 Markdown」批量导入本地 `.md` 文件
3. 在左侧笔记列表中选择笔记即可开始编辑
4. **模式切换**：顶部「记事本模式」/「MD 模式」按钮
   - **记事本模式**：所见即所得编辑器，直接在可视化界面中排版
   - **MD 模式**：左侧 Markdown 源码编辑 + 右侧实时预览，支持双向滚动同步
5. **工具栏隐藏**：点击右上角工具栏图标按钮，收起/展开工具栏
6. **排版操作**：通过工具栏设置字体、字号、粗体、斜体、颜色、高亮、标题、列表、对齐、图片、分隔线、标签等
7. **侧边栏调节**：拖拽侧边栏右边框可调整列表宽度
8. 所有内容自动保存，无需手动操作
9. 底部标签区域可输入标签名筛选笔记

### 设置

1. 点击侧边栏底部的「设置」按钮打开设置弹窗
2. **主题颜色**：从 4 个预设主题卡片中一键切换，切换后调色板自动同步
3. **自定义调色板**：通过 5 个颜色选择器自由调整配色
4. **保存预设**：点击左侧按钮将当前配色保存为自定义预设（最多 5 个）
5. **选择预设**：点击右侧下拉框查看、应用或删除已保存的配色方案
6. **语言切换**：中文 / English 一键切换，立即生效

## 数据存储

所有用户数据保存在本地，不会上传到任何服务器：

```
%APPDATA%/com.pytsingtodo.app/data/
├── todos.json          # 待办事项数据
├── fonts.json          # 自定义字体注册表
├── settings.json       # 主题、语言、自定义颜色、已保存预设
└── notes/
    ├── index.json      # 笔记元数据索引（标题、标签、时间）
    └── <uuid>.md       # 笔记内容文件
```

> 💡 如需备份数据，复制上述目录即可。更换电脑时，将该目录复制到新电脑对应位置即可恢复所有数据。

## 常见问题

**Q：数据存储在哪里？卸载后会丢失吗？**
A：数据存储在 `%APPDATA%/com.pytsingtodo.app/data/` 目录下。卸载应用不会自动删除该目录，但建议提前备份。

**Q：支持哪些操作系统？**
A：目前仅支持 Windows 系统。

**Q：笔记支持什么格式？**
A：记事本模式下为富文本（WYSIWYG），MD 模式下为 Markdown。两种模式可自由切换，内容会自动转换。

**Q：如何导入已有的 Markdown 文件？**
A：在记事本页面点击「导入 Markdown」按钮，选择 `.md` 文件即可批量导入。
